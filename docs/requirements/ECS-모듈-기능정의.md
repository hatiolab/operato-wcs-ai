## 1. Put-To-Light (PTL) — 출고/반품 분류 시스템

랙과 셀, 표시기 기반으로 **B2B(매장별) / B2C(주문별)** 분류를 처리하는 시스템입니다.
출고와 반품 모두 처리 가능합니다.
