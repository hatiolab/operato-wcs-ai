## ⚠️ **중요: 즉시 해결 필요 항목**

### ❌ Java 버전 불일치 (빌드 실패 원인)

**문제**:
- `build.gradle` 명시: Java 17 필요
- Spring Boot 3.2.4: Java 17 이상 필수
- **현재 시스템**: Java 8 사용 중

**해결 방법**:
```bash
# Java 17 설치 (SDKMAN 사용 권장)
curl -s "https://get.sdkman.io" | bash
sdk install java 17.0.9-tem

# 또는 Homebrew
brew install openjdk@17

# Java 17로 전환 후 빌드 재시도
./gradlew clean build
```

이 문제를 해결하지 않으면 **개발을 시작할 수 없습니다**.
